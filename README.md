# Elementum

A pixel-close React implementation of the Elementum Figma design assignment.

## Live Demo
🔗 https://elementum-chi.vercel.app/

## Tech Stack
- React 18
- CSS Modules
- Google Fonts (Syne + Space Grotesk)

## Sections
- Navbar (fixed, scroll-aware, responsive hamburger menu)
- Hero (display headline with highlights, organic avatar scatter)
- About (zigzag layout, circular images, decorative triangles)
- Services (table-style rows with hover interactions)
- Testimonials (floating avatars, quote card)
- Newsletter (mint background, subscribe button)
- Footer (4-column grid)

## Features
- Fully responsive (mobile, tablet, desktop)
- Scroll reveal animations
- Hover states on all interactive elements
- Subtle entrance animations

## Getting Started

```bash
npm install
npm start
```

## Deployment
Deployed on Vercel via GitHub integration.
